A V2 chocolate-themed memory game built with Godot 4.6 (GDScript). Built to make example of an AdverGames.
Memorize where the chocolates appear on a 4x4 grid, then click the correct tiles before time runs out to earn discounts.
View on [@github](https://github.com/kaeri-gg/ferrero-advergame).
